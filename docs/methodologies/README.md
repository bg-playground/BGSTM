# Testing Methodologies

This directory contains comprehensive guides for implementing software testing practices across different software development methodologies.

## Available Methodologies

### [Agile Testing](agile.md)
Comprehensive guide for testing in Agile development environments, focusing on continuous testing, collaboration, and iterative delivery.

### [Scrum Testing](scrum.md)
Detailed framework for testing within Scrum sprints, including sprint planning, daily activities, reviews, and retrospectives.

### [Waterfall Testing](waterfall.md)
Complete guide for testing in Waterfall projects, covering phase-gate approaches and comprehensive documentation.

### [Methodology Comparison](comparison.md)
Side-by-side comparison of testing approaches across different methodologies to help you choose the right approach.
